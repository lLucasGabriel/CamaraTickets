## Pages

7 pages:

- **Login:** The page to login and access the site.
- **New Ticket:** The page that has the function of adding a new ticket on the Table.
- **Manage Tickets:** The page that has the functionality to manage tickets.
- **New Member (Private):** The page that has the functionality to register users on the site.
- **Profile (Private):** The page that presents the information (name, age, origin, email and avatar) of the registered user, which has the functionality to edit them.
- **Manage Members:** The page that has the functionality to manage users.
- **Monitor service:** The page that has the functionality to answer and close tickets.
